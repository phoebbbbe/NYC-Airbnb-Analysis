# 2019 年紐約市 (NYC) Airbnb 房源數據分析

## 1. 背景與概述
Airbnb 自 2008 年創立以來，憑藉其創新的共享住宅模式，成功建構了連結全球房東與旅客的橋樑，提供深具在地特色的住宿體驗。其中，紐約市 (NYC) 作為全球觀光與商業樞紐，擁有極其龐大且複雜的房源生態系統，本專案透過對 2019 年紐約市 Airbnb 公開房源數據的系統性分析，挖掘隱藏在定價、區域背後的關鍵洞察，為房東與平台運營者提供數據驅動的決策支持。

* 主要工具：Python (Pandas, Matplotlib, geopandas, seaborn)

## 2. 數據結構與預處理
此分析使用數據包含約 48,000 筆房源資料，涵蓋 16 個欄位：
| 類別 | 關鍵欄位 | 說明 |
|---|---|---|
| **房源資訊** | `id`, `name`, `room_type` | 包含房名稱及類型 |
| **房東資訊** | `host_id`, `host_name`, `calculated_host_listings_count` | 房東相關資訊 |
| **地理資訊** | `neighbourhood_group`, `neighbourhood`, `latitude`, `longitude` | 涵蓋紐約五大行政區 |
| **定價與條件** | `price`, `minimum_nights`, `availability_365` | 房源核心經濟指標 |
| **評論指標** |   `reviews_per_month`, `number_of_reviews`, `last_review` | 用戶活躍度指標 |

預處理過成包括：
* 填補缺失值。
* 過濾離群值。
* 過濾殭屍房源。

### 房源供給地理分佈

透過地理熱力圖，可以觀察到 2019 年紐約市 Airbnb 市場高度集中在**布魯克林 (Brooklyn, 42.3%)** 與**曼哈頓 (Manhattan, 41.6%)**，兩區合計甚至佔據了全紐約超過 80% 的房源供給。相較之下，史泰登島 (Staten Island) 僅佔 0.9%，顯示出市場重心極度偏向商業與文化機能發達的核心區域。

從分區供給量排行，進一步分析前30名的地區，發現布魯克林區呈現顯著的集中趨勢。其中，**Bedford-Stuyvesant** 是布魯克林區也是全紐約供給量最多的地區，**擁有 2,736 筆房源（佔比達 20.5%）**，緊隨其後的是 **Williamsburg，擁有 2,384 筆房源（佔比 17.9%）**，兩者合計撐起了布魯克林區近**四成的市場供給**。曼哈頓區則以 **Harlem (1,900 筆, 14.5%)** 居首。值得注意的是，曼哈頓的供給分佈較布魯克林更為平均。

### 房源價格箱型分佈

從價格箱型分佈圖中，很明顯的觀察到供給量前30名的地區中，曼哈頓區的價格分佈最廣也最高，然後是布魯克林區，最後是皇后區。

布魯克林區中位價最低的區是 Bushwick (供給量佔比 12.3%)，屬於下層中產階級社區；最高的地區是 Park Slope，是歷史悠久的高級住宅區 (供給量佔比 2.3%)。房源最多的 Bedford-Stuyvesant 中位價位於整體價格中間偏低。
曼哈頓區南部的 Midtown、Chelsea、West Village 與 Financial District，不僅中位價顯著高於其他區域，價格區間也更廣，且存在大量的極端價格點。其中，Financial District 與 West Village 的 Lower Quartiles，明顯高於紐約市的大多數社區。

### 各區域房型結構

透過各行政區房源類型的比例分佈，觀察到供給量前30名的地區在「產品定位」上的顯著差異。

* 曼哈頓區是唯一「整套房源 (Entire home/apt)」佔比過半的區域，高達 57.9%。
* 布魯克林區的房源結構最為均衡，「獨立房間 (Private room)」與「整套房源」分別佔比 50.3% 與 47.7%，是近乎 1:1 的分佈。
* 皇后區與布朗克斯，超過六成的供給來自「獨立房間」。

<img width="600" src="/img/img03.jpg">

* 曼哈頓區中 West Village 與 Financial District 的「整套房源 (Entire home/apt)」佔比最高，分別為 75.3% 與 85.0%。然而位於北邊的 Washington Heights 其「整套房源」僅 31.6%。
* 布魯克林區中 Bushwick 的「獨立房間 (Private room)」佔比高達 68.7%。另外，雖然該區結構較均衡，但 Park Slope 的「整套房源」佔比卻高達 71.9%。

### 各區域房型結構與定價關聯分析

觀察三種房型在供給量前30名的地區，中位數的變化折線圖，發現市場呈現明顯的階梯式定價：「整套房源 (Entire home/apt)」遠高於「獨立房間 (Private room)」，而「合住房間 (Shared room)」價格則是最低的。

其中，曼哈頓的 Financial District 展現了極其顯著的趨勢，「整套房源」的中位價最高（$223），且該區的「合住房間」也為最高（$150），超越了「獨立房間」甚至多數地區「整套房源」的價格。

### 各區域房源價格水平與市評論熱度關聯分析

此折線圖比較了供給量前30名的地區中，房源的價格中位數（紅色折線）與評論數中位數（黃色折線）的變化和關係。趨勢顯示，兩條折線在多個區域呈現顯著的反向變動，反映出高價格的房源，評論數往往較低，而低價格的房源，評論數也往往較高。

其中，皇后區的 Flushing 展現了極其突出的趨勢——該區擁有最低的價格中位數（約 $60），卻對應著最高的評論中位數（25 筆以上）。

## 4. 洞察
* 隨著地理位置逼近曼哈頓核心商業區，房源結構明顯趨向整套房源，反映出曼哈頓房東傾向以高單價、高隱私的整套房源，滿足商務與高端旅客的需求。
* 緊鄰曼哈頓的布魯克林區，呈現出「過渡型市場」的特徵，整套房源與獨立房間的供給比例均衡，具有極高的市場彈性：既能承接曼哈頓外溢的中產階級居住需求，又能吸納追求在地文化體驗與性價比的年輕客群。
* 而遠離曼哈頓核心的皇后區，超過六成的供給集中於獨立房間，雖然單價較低，但創造了高周轉率，成為市場中「長尾需求」的主要承載地。
